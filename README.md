# CashApp

A UI clone of some of CashApps views built with SwiftUI. This project is primarily for education purposes.


# Requirement 
- Xcode 12.4 or up 
- SwiftUI support


# Design source 
- https://mobbin.design/apps/cash-app


# Views 
- [ ] Phone number Onboarding
- [ ] Invite Contacts 
- [ ] Profile 
- [ ] Investing 
- [ ] Send & Request Money
- [ ] Cash Card
- [ ] Banking
- [ ] QR Code scanner
